# Проект: Место

## Описание проекта
Проект **Место** является третьим проетом в курсе. В этом проекте используется адаптивно-отзывчивая верстка, Grid и Fleks технологии, а также используются формы и input элементы.

### Используемые технологии:
* HTML5
* CSS3
* Flexbox
* Grid-layout
* Адаптивная верстка
* PixelPerfect

## Требования:
* HTML5
* CSS3
