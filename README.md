## Описание проекта
Проект **Место** это невероятный сайт. В этом проекте используется адаптивно-отзывчивая верстка, JavaScript классы, WebPack

### Используемые технологии:
* JavaScript
* HTML5
* CSS3 (Flexbox, Grid-layout, Адаптивная верстка, PixelPerfect)
* WebPack

## Требования:

* CSS3
* JavaScript
* WebPack
* NodeJS
